# 人 — 辩论者

本目录记录辩论参与者的角色定义、能力进化和协作关系。

## 五位辩手

| 角色 | 文件 | 职责 | 核心能力 |
|---|---|---|---|
| 🔍 话题搜集者 | `agents/roles/collector.md` | 搜索争议话题、审核打分、精炼表述 | 八维度评分（争议性/可验证性/分层空间/知识价值/红旗密度/隐匿密度/异常聚类密度/品牌化程度） |
| 🔴 质疑者 | `agents/roles/challenger.md` | 找漏洞、红旗、动机、知识谱系、隐匿模式 | 28条红旗信号、7种异常聚类识别、7种隐匿模式识别、防御模式检测 |
| 🟢 推理者 | `agents/roles/reasoner.md` | 证据分级、归属阶梯、最强解释、概率聚合 | 八步核查、S⁻级证据、完美犯罪处理层级、科学共识污染分析 |
| ⚖️ 裁判者 | `agents/roles/judge.md` | 分层结论、概率聚合、最终报告 | 12种判断标签、隐匿评估、可知边界、辩论质量评估 |
| 🛠️ Skill优化者 | `agents/roles/optimizer.md` | 审查辩论、沉淀方法论、同步角色能力 | 6种方法论增量类型、5维案例评分、角色能力同步检查清单 |

## 能力进化轨迹

角色能力随 skill 方法论同步进化。每次 SKILL.md 更新后，相关角色在 24 小时内注入新方法论武器。

| Challenger | 异常聚类识别清单（7种）、隐匿模式识别（7种）、防御模式检测、制造传统检测、科学共识污染检测、红旗从7条→28条 |
