# 辩论结论

## Aristotle 品牌语料库

**结论**：真实人物核心(~20%) + 学派教学材料 + Andronicus重组 + 阿拉伯/拉丁翻译改写 + 经院哲学权威品牌 + Bekker现代正典 + 教材英雄神话。

| 层 | 判断 | 置信度 |
|---|---|---|
| 人物存在 | 已确认 | 极高 |
| 核心教学材料归属 | 高度可能 | 高 |
| 全部著作独自完成 | 过度宣称 | 高 |
| 后世"百科全书式天才"叙事 | 品牌化 | 极高 |

**整体可信度**：中高

| 层 | 判断 | 置信度 |

| 全部剧作独自完成 | 过度宣称 | 高 |
| 后世"孤独天才"叙事 | 品牌化 | 已确认 |

**整体可信度**：中高

## 戴比尔斯钻石"传统"

**结论**：钻石订婚戒指是戴比尔斯20世纪系统性营销制造的人造传统，不是古老文化习俗。

| 层 | 判断 | 置信度 |
|---|---|---|
| "古老传统"宣称 | 已证伪 | 极高 |
| 戴比尔斯制造习俗 | 已确认 | 极高 |
| 消费者"自由选择" | 过度宣称 | 高 |
| 钻石订婚戒指作为文化规范 | 品牌化 | 高 |

**整体可信度**：极高（事实层）/ 中（哲学层）

## 制糖业操纵营养科学

**结论**：制糖业通过资助研究+审阅论文+压制异见+渗透政策，系统性将营养科学推向"脂肪坏、糖好"方向。

| 层 | 判断 | 置信度 |
|---|---|---|
| 制糖业付费操纵研究 | 已确认 | 极高 |
| 操纵影响膳食指南 | 高度可能 | 高 |
| "脂肪导致心脏病"宣称 | 过度宣称 | 高 |
| "糖相对无害"宣称 | 已证伪 | 高 |
| 营养科学"共识" | 品牌化 | 高 |

**整体可信度**：极高（操纵事实）/ 中（完整影响范围）

## 斯坦福监狱实验

**结论**：实验不是科学实验，而是研究者参与并引导的戏剧表演。BBC重复实验得出相反结论。

| 层 | 判断 | 置信度 |
|---|---|---|
| 实验作为"情境力量"证据 | 已证伪 | 极高 |
| 研究者干预实验 | 已确认 | 极高 |
| 参与者被引导表演 | 已确认 | 高 |
| 教材中的"经典"地位 | 品牌化 | 高 |

**整体可信度**：极高（实验被操纵）/ 中（"情境力量"理论本身）

## 判断标签说明

| 标签 | 含义 |
|---|---|
| 已确认 | 多源独立验证，S/A级证据充分 |
| 高度可能 | 证据链完整，虽有缺口但整体可信 |
| 过度宣称 | 宣称强度远超证据支撑 |
| 品牌化 | 后世将混合/被操纵的材料归到权威名字下 |
| 已证伪 | 有明确反证 |
