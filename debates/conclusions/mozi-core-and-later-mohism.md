# 《墨子》中哪些篇章确为墨子本人思想，哪些是后期墨家的发展？

**日期**：2026-06-20 | **轮次**：3轮

## 结论

《墨子》不是墨子一人的完整著作，而是墨家学派语料库。兼爱、非攻、尚贤、尚同等较可能保存早期墨子或早期墨家核心；《经》《经说》《大取》《小取》等更像后期墨家名辩和认识论发展；城守诸篇体现组织技术实践。

## 分层判断

| 层 | 判断 | 置信度 |
|---|---|---|
| 墨子人物 | 高度可能 | 高 |
| 早期核心思想 | 高度可能 | 高 |
| 名辩篇本人亲作 | 过度宣称/待证 | 中高 |
| 后期墨家发展 | 高度可能 | 高 |
| 全书个人著作 | 过度宣称 | 高 |

## 核心发现

- 战国诸子互相批评支撑墨家显学存在。
- 内部重复篇组显示讲学模板和集体编纂。
- 技术化名辩与城守材料反映后期学派发展。
