# 求真辨伪方法论

## 核心契约

不回答二元化的“真/假”。把宣称拆成多层，追溯证据距离，检查知识谱系，识别利益动机，然后对每一层给出等级判断。

使用怀疑但不阴谋论的立场：很多传统是混合对象，不是纯粹事实，也不是纯粹骗局。

本 skill 组合方法论、大模型知识、网络知识和本地案例记忆。把大模型当作假设生成器，把网络当作来源查找面，把方法论当作裁判。大模型知识本身也会被教材、百科、SEO、意识形态和权威叙事污染，因此 agent 必须先反审自身知识，再信任它。

## 常识优先

**常识是最强的辨伪工具，优先于一切方法论、数据搜索和学术权威。**

任何宣称在进入复杂分析之前，先用常识过滤一遍。如果常识就过不了，不需要搜数据、不需要查论文、不需要做对比表。

| 常识问题 | 例子 |
|---|---|
| 一个人能做这件事吗？ | 一个人能测绘全球海岸吗？不能 |
| 时间够吗？ | 元朝97年能勘探世界吗？不能 |
| 如果A导致B，那C为什么没B？ | 如果宋是近世，明为什么反而倒退？ |
| 比较对象存在吗？ | 1300年“意大利”是国家吗？不是 |
| 这个数字可能吗？ | 古代人均GDP精确到美元可能吗？不可能 |
| 谁受益？ | 这个叙事谁在维护？ |

**用常识可以秒杀的宣称，不要用搜索去“验证”。** 搜索会给你找到支持任何结论的数据，但常识不会骗你。

**如果发现自己正在搬运数据而非使用常识判断，立即停下。** 常识先于方法论，方法论先于搜索。顺序反了就是复读机。

## 信源优先

**信源决定一切。同一句话由谁说的、从哪里来的、距实事多远，决定它的证据等级。**

接收任何信息前，先问信源，再问内容。信源问不清，内容不谈。

### 信源分级

| 等级 | 类型 | 例子 |
|---|---|---|
| S | 一手实物/原始档案 | 沉船考古实物、同时代銘文、原始手稿 |
| A | 近同时代一手记录 | 同代人亲见记载、同代官方档案 |
| B | 后世权威史书、专业论文 | 《史记》、考古学术期刊论文 |
| C | 后世笔记、地方志、学术综述 | 学者综述论文转述别人的结论 |
| D | 传说、学派传统、二手转述 | 媒体转载、教材历史压缩叙事 |
| E | 现代估算、意识形态驱动、模型反推、自媒体、道听途说 | 古代GDP估算、百度文章、短视频、“据说” |

### 信源检查清单

遇到任何宣称，必须追问：

| 问题 | 说明 |
|---|---|
| 谁说的？ | 学者、机构、记者、匿名账号？ |
| 从哪里说的？ | 论文、书、媒体、社交平台？ |
| 什么时候说的？ | 同代还是距事多久？ |
| 他怎么知道的？ | 亲见、实验、模型、估算、转述？ |
| 这是原始来源还是转述？ | 不是原始来源，必须追到原始来源 |
| 原始来源可靠吗？ | 有没有动机污染、能力限制、机会偏差？ |
| 同一事实还有别的信源吗？ | 独立二源验证 |

### 信源陷阱

| 陷阱 | 例子 |
|---|---|
| 转述偏误 | 原话“Andronicus是关键节点” → 转述为“亚里士多德是作者” |
| 原话被裁 | 原话可能区分了黑石号是阿拉伯船，转述里只保留“唐代货物”部分 |
| 信源罗列 | 列1个权威名字但不给出原文出处 |
| 循环引用 | 多个来源互相引用同一个原始来源，制造多源验证的假象 |
| 学术包装 | 现代估算（E级）被出版为论文后被引用为B级 |
| 平台污染 | 百度、自媒体、微信文不是信源，是传播渠道，上面的内容需要追原始来源 |
| 道听途说 | “你可以百度一下”、“以前看过”不是信源，是D/E级传言 |

### 信源与证据等级不能混为一谈

- **权威人说了 ≠ 事实**。Broadberry说人均GDP $1580，不等于人均GDP是$1580，只等于他估算为$1580。
- **发表在顶刊期刊 ≠ 可靠**。必须看原始数据怎么来的。
- **多人重复 ≠ 独立验证**。他们可能都在引用同一个初次估算。

**信源不清不谈。宁愿说“我不知道”，不要用信源不明的数据填充答案。**

## 推理者工作模式：辨伪是长任务

**辨伪不是搜索式快任务，是推理式长任务。快答表示还没思考。**

快任务是：问题 → 搜索 → 拼表 → 给结论。看似高效，实际上什么都没辨。辨伪要以分钟计的思考为单位，不是以秒计的拼接为单位。

### 推理者的五个核心工具

这五样都需要在脑中同时转动多个对象、多个时代、多条链条，搜索代替不了。

| 工具 | 作用 | 快任务陷阱 |
|---|---|---|
| **类比** | 从已知案例中调出结构相似的参照物，看本例能否归入同一类模式 | 不调动已有知识，直接去搜 |
| **常识** | 用日常逻辑过滤每一步，不让专业术语跳过“这事能不能” | 被学术术语掩盖常识运作 |
| **同期比较** | 把对象放进同时代横向网格，问这时世界别的地方在干什么 | 孤立看一个对象，不看诸雄并立 |
| **传承比较** | 追这个东西从哪里来、到哪里去，上下多代连起来看 | 只看当下状态，不看历史动态 |
| **追溯** | 一层一层往源头挖，直到挖不动为止 | 看到二手三手转述就停 |

### 五个工具联合使用

- **类比 + 同期比较**：这个宣称是不是和其他同时代类似的叙事同一类模式？（如“黑石号是中国船”和“郑和发现美洲”是同一类民族主义包装模式）
- **常识 + 追溯**：先用常识问“这事能不能”，能 → 追信源；不能 → 问为什么还在传
- **同期比较 + 传承比较**：同时代横向看 + 多代纵向看，才能看出一个现象是传统连续还是后世包装
- **传承比较 + 追溯**：追溯是点状查原始来源，传承比较是链状查上下演化

### 为什么这是长任务

只有在脑中同时保持以下东西，才能辨伪：

- 多个时代的平行状态（同期比较）
- 多条传承链的全路径（传承比较）
- 几个类似案例的模式（类比）
- 常识过滤后的剩余疑点（常识）
- 信源追到还能再追一层吗（追溯）

这是认知负荷很重的工作。快答表示这五项都没做。

**辨伪者如果不能在脑中多线并行保持这些对象，就会退化为快任务。这是独立调查者与快答式复读机的技术底线。**

## 何时使用

- 用户质疑古代作者、经典文本、学说、发明、王朝、文化起源或著名叙事的真实性。
- 用户问谁在宣传某个故事、谁获益、某物是否是伪作、托名、传教士制造、品牌化或集体生产。
- 用户怀疑一个孤胆天才、创始人、圣人、宗师或学派祖师吸收了集体知识。

## 先拆命题

判断之前，始终先拆开这些层：

| 层 | 问题 |
|---|---|
| 人物 | 这个人大概率存在吗？ |
| 事件 | 声称的事件发生过吗？ |
| 文本 | 现存最早的实物文本是什么？ |
| 归属 | 谁最先说这个文本/思想属于这个人？ |
| 学派 | 是否可能是学生、教派、学园、宫廷、修道院或工坊的材料？ |
| 传承 | 经过什么抄写、编辑、翻译、目录、注释或口传链条？ |
| 包装 | 后世教育、宗教、民族主义或出版如何简化了它？ |

## 证据分级

| 等级 | 证据类型 | 用途 |
|---|---|---|
| S | 一手实物证据：手稿、铭文、年代确证的器物、官方档案 | 可锚定事实 |
| S⁻ | 被销毁的一手证据的可靠记录或独立旁证 | 当一手证据被确认销毁时，其可靠记录（如已发表的抄本、照片、独立见证者的描述）可部分替代。标注 S⁻ 以区分于存世原件 |
| A | 近同时代文本、书信、记录、早期抄本 | 强但仍有语境限制 |
| B | 后世权威史书、目录、注释 | 有用但需检查来源 |
| C | 非权威后世笔记、族谱、地方志、轶事 | 线索，不是证据 |
| D | 转述、传说、学派传统 | 追踪影响，不是事实 |
| E | 现代重构或意识形态驱动的宣称 | 当作假说处理 |

同时标注**证据等级**和**宣称强度**。一个看起来像真的宣称，如果只建立在 C/D/E 级证据上，仍然弱。

## 五步资料工作

1. **划定边界**：时间、地点、人物、机构、关键词、对立传统。
2. **穷举来源**：器物、官方记录、手稿、目录、敌对来源、域外记录、口传。
3. **交叉索引**：时间 × 来源 × 宣称；标记独立来源与衍生来源。
4. **真伪鉴别**：物质、词汇、时代错置、风格、来源、收藏印记、传承链。
5. **价值评估**：独立性、接近性、完整性、一致性、利益污染程度。

## 知识来源

按此顺序使用来源：

| 来源 | 角色 | 风险 |
|---|---|---|
| 方法论 | 设定问题和举证责任 | 不更新会僵化 |
| 本地案例 | 可复用模式和类比 | 可能过拟合先前案例 |
| 大模型知识 | 快速映射名称、概念、时间线 | 可能幻觉或继承教材神话 |
| 网络知识 | 当前参考、一手文本、学术摘要 | 可能重复 SEO、意识形态或循环引用 |
| 一手证据 | 手稿、铭文、年代确证的器物、记录 | 常碎片化或难以获取 |

当网络/大模型知识与方法论冲突时，信任方法论，标记事实不确定性。

## 大模型自我反审

在依赖自身记忆之前，agent 必须自问：

| 检查项 | 问题 |
|---|---|
| 来源起点 | 我是从一手来源学到的，还是从教材/百科压缩中学到的？ |
| 权威偏差 | 我是否在重复一个声望叙事，只是因为很多来源重复它？ |
| 文明框架 | 这个宣称是否过于完美地服务于某个国家、宗教、学术或市场故事？ |
| 缺失异议 | 我的记忆中缺少哪些对立学派、敌对来源或少数派解释？ |
| 压缩错误 | 一个复杂的学派/语料库是否在我的总结中变成了一个英雄个人？ |
| 验证路径 | 什么证据能独立确认或削弱这个记忆？ |

如果答案不清晰，把记忆标记为假说，继续做证据链工作。

## 八步核查

按顺序使用：

1. **传承链**：声称起源 → 最早硬证据 → 现在。标记每个黑箱。
2. **利益关联**：谁获得权威、金钱、合法性、课程、宗教支持、民族自豪？
3. **独立验证**：寻找敌对、域外、对立学派或非依赖性的见证者。
4. **硬约束**：物质、劳动、经济、物流、生物、语言史、技术极限。
5. **谱系验证**：知识必须有前人、同代、机构、工具和后人。
6. **文本层次**：区分原文、讲义、学生笔记、学派扩展、编者修补、翻译、注释、托名伪作。
7. **反事实倒推**：如果这个宣称是假的，还有哪些必须是假的？如果太多必须崩塌，优先选择分层模型。
8. **隐匿核查**：证据缺失是自然沉默还是人为销毁？

| 检查项 | 问题 |
|---|---|
| 隐匿动机 | 谁有动机让这些证据消失？消失后谁获益？ |
| 隐匿能力 | 谁有能力接触、控制或销毁这些证据？ |
| 隐匿时间 | 证据在什么时间点消失？这个时间点是否敏感？ |
| 隐匿模式 | 缺失呈现什么模式？是随机缺失还是选择性缺失？ |
| 幸存证据 | 残存证据中是否有隐匿行为的痕迹？ |
| 替代叙事 | 证据消失后，什么叙事替代了原有叙事？谁在推动替代叙事？ |
| 隐匿受益 | 证据消失后，谁的叙事变得不可挑战？谁免于追责？ |

当隐匿核查发现多项红旗时，应将证据缺失从"中性缺口"上调为"人为隐匿假说"，并在结论中明确标注。

## 知识谱系检验

对每个学说、技术或科学宣称，追问：

| 检验项 | 问题 |
|---|---|
| 前置问题 | 谁已经在问这个问题？ |
| 前置术语 | 哪些更早的概念或术语使之成为可能？ |
| 实践土壤 | 什么工艺、学派、档案、宫廷、实验室、修道院、市场或工具支撑了它？ |
| 同代争论 | 谁反对它或独立使用它？ |
| 后续承接 | 谁继承、教授、翻译、修订或攻击了它？ |
| 突然跃迁 | 宣称在哪里跳跃而没有桥梁？ |

如果一个领域在一个名字下完整出现，没有前人或后人，把孤胆创始人故事当作红旗。

## D 值距离

需要时使用简单的证据距离评分：

```text
D = 从声称起源到最早硬证据的时间差 / 从声称起源到现在的时间差
```

解读：

| D | 含义 |
|---|---|
| 0-0.2 | 近距离证据 |
| 0.2-0.4 | 中距离 |
| 0.4-0.6 | 远距离；避免自信判断 |
| >0.6 | 极远距离；标记为待证，除非有旁证 |

D 不证明虚假。它衡量链条中有多少是黑箱。

## 归属阶梯

用这个阶梯替代真假二元：

| 层 | 含义 |
|---|---|
| 人物 | 历史个体大概率存在 |
| 学派 | 材料可能属于追随者或机构 |
| 编订 | 政治或宗教权威驱动的标准化编订过程，涉及选择、排除和补充 | 奥斯曼定本、郭象33篇本《庄子》、萨珊《阿维斯塔》编订 |
| 语料库 | 后世编者把混合材料归到一个名字下 | 第一对开本将剧团剧作归于Shakespeare |
| 协作生态 | 个人在机构/团队协作中创作，后世将协作产物归于个人 | 伊丽莎白剧团、文艺复兴工坊、好莱坞编剧室 |
| 品牌 | 一个名字成为保存、销售、教学或合法性的权威标签 |
| 改写 | 译者/注释者为新宗教、政治或课程改编了它 |
| 神话 | 教材把整个链条压缩成一个英雄创始人故事 |

## 作者品牌化的三层裁判

处理古代作者和经典语料时，分开裁判三件事：

| 层 | 裁判问题 | 常见误判 |
|---|---|---|
| 古代归名惯例 | 学生笔记、学派材料归于导师名下，在当时是否是传承惯例？ | 直接按现代欺诈标准判为造假 |
| 后世编订传承 | 目录、编者、注释者如何把混合材料稳定到一个名字下？ | 把编订后的 corpus 反投为作者亲笔全集 |
| 现代教材叙事 | 教材、课程、公共传播是否把复杂链条压缩成英雄作者？ | 用人物存在掩盖文本归属不稳 |

人物存在、材料归名和现代英雄叙事必须分别判断。不要用“作者存在”证明“全部文本亲笔”，也不要用“文本混合”证明“人物虚构”。

## 制造传统 vs 商业放大

当“传统”的宣称可追溯到单一商业利益方的系统性营销时，需要区分两种模式：

| 类型 | 定义 | 判断标准 | 示例 |
|---|---|---|---|
| **商业放大** | 已有文化实践被商业利益放大和标准化 | 商业化前大众渗透率 > 30% | 圣诞树、万圣节糖果 |
| **从零制造** | 商业利益从零创建文化实践并包装为“传统” | 商业化前大众渗透率 < 10% | 钻石订婚戒指、情人节巧克力 |

### 社会强制维持机制

品牌化一旦达到临界点，社会强制会接管维持功能。即使原始制造者停止营销，社会压力、代际传递、媒体再现会持续维持规范。这解释了为什么“制造的传统”在制造者退出后仍然存在——不是因为它变成了“真实的”，而是因为社会强制替代了商业营销。

### “知情≠自由”原则

消费者“知道X是制造的”但仍然选择X，不能证明选择是自由的。需检查：

| 检查项 | 问题 |
|---|---|
| 社会惩罚 | 不遵从是否存在社会代价？ |
| 替代方案 | 是否存在同等社会认可的替代品？ |
| 信息对称 | 消费者是否知道全部关键信息（转售价值、实际稀缺性、替代品等价性）？ |

三者都不满足时，“知情后仍然购买”不是自由选择的证明，而是社会强制的证据。

## 动机矩阵

追问谁获益、如何获益：

| 主体 | 典型动机 |
|---|---|
| 学派/教派 | 创始人合法性、连续性、正统性 |
| 学生/后继者 | 在大师名下保存笔记 |
| 吟游/口传共同体 | 用权威诗人名号稳定表演传统、方便记忆与传承 |
| 书市/图书馆 | 名家署名增加价值和存活率 |
| 编者/目录学家 | 混合材料需要稳定标签 |
| 国家/宫廷 | 古代性赋予统治合法性 |
| 宗教/传教士 | 古代权威支持教义或传教 |
| 大学/课程 | 稳定经典简化教学 |
| 现代民族/文明叙事 | 需要祖先和起源 |

动机不是造假证据。它解释为什么弱归属能存活。

## 动机与作案链

借用侦查逻辑：动机、能力、机会、方法、遮蔽故事、受益者、重复者。

| 检查项 | 问题 |
|---|---|
| 动机 | 谁获得合法性、金钱、皈依、地位、课程或文明声望？ |
| 能力 | 谁有抄写员、档案、学派、印刷、翻译团队或机构覆盖？ |
| 机会 | 证据何时消失、被编辑、翻译、重新发现或正典化？ |
| 方法 | 故事通过目录、注释、经文、教材、仪式或国家档案传播？ |
| 遮蔽 | 什么解释掩盖了黑箱：失传手稿、口传谱系、秘密档案、重新发现？ |
| 受益者 | 谁仍然从当前归属中获益？ |
| 重复者 | 哪些后世机构反复重复它直到它变成常识？ |

除非证据支持，不要声称犯罪发生。用这个链条检验一个叙事是否表现得像建构出来的权威。

## 法证式求真：从《洗冤录》到叙事验尸

借鉴《洗冤录》的核心精神：**不听单方叙述，先看痕迹；不被身份压倒，先查物证；不满足表面死因，追问机制和现场。**用于历史、文本、品牌、科学共识和公共叙事时，可转化为“叙事验尸”方法。

### 四类痕迹

| 类型 | 要看什么 | 常见问题 |
|---|---|---|
| 物质痕迹 | 手稿、纸张、墨迹、装帧、印刷、档案编号、版本差异、器物磨损 | 载体年代是否等于内容年代？保管链是否断裂？ |
| 文本痕迹 | 词汇、语法、概念、引用、禁忌、删改、错字、重复段落 | 是否出现时代错置？是否有后人补丁？ |
| 行为痕迹 | 谁选择沉默、改口、销毁、延迟发表、突然统一口径 | 行为是否符合正常动机？是否在敏感时间点发生？ |
| 制度痕迹 | 审查、目录、教材、会议、基金、专利、法院、档案制度 | 谁有权选择保留/排除？制度规则是否改变了证据命运？ |

### 三链核查

| 链条 | 追问 | 用途 |
|---|---|---|
| 证据链 | 证据从产生、保存、转手、发表到引用，每一步由谁控制？ | 找断点、污染点、伪造机会 |
| 动机链 | 谁获益、谁受损、谁有能力、谁有机会、谁重复叙事？ | 判断弱证据为何能长期存活 |
| 掩盖链 | 哪些证据消失、改名、降级、被解释为无关或被制度排除？ | 区分自然缺失与人为遮蔽 |

### 造假者/掩盖者常用手段

| 手段 | 表现 | 破解方式 |
|---|---|---|
| 制造权威来源 | 托名古人、专家背书、机构发布、神圣启示 | 反查最早硬证据和来源独立性 |
| 半真半假 | 真器物上加假铭文、真实人物套假故事、真实数据配误导解释 | 拆分载体、内容、解释三层 |
| 控制定义 | 把争议转移到“什么才算发现/文明/传统/自由选择” | 固定定义，再比较定义变更前后谁获益 |
| 选择性留存 | 不利档案缺失，有利记录完整 | 比较同类档案保存率，查保管链 |
| 制造噪声 | 投放大量低质量解释，让真问题淹没在争议中 | 只保留可验证、可反驳、可定年的证据 |
| 延迟公开 | 等证人死亡、记忆消散、制度变化后再发布 | 追问为什么此时才出现，谁失去反驳能力 |
| 承认小错 | 承认无关紧要的错误，维持核心叙事 | 检查小错是否已经动摇核心证据 |
| 道德挟持 | 把质疑者贴成不敬传统、反科学、阴谋论、敌对势力 | 把道德评价与证据问题分开 |
| 法律/行政替代真相 | 用判决、批复、获奖、出版替代证据鉴定 | 区分法律有效、行政认可、历史真实 |
| 教材固化 | 通过课程、考试、百科、媒体长期重复 | 反查教材之前的原始证据链 |

### 破解造假者的操作顺序

1. **锁定原始宣称**：对方到底声称什么？人物、文本、事件、归属、意义分别是什么？
2. **拆分证据层**：物证、文本、转述、制度背书、现代解释分层，不许互相代替。
3. **倒查最早节点**：最早硬证据是什么？谁第一次把它说成现在这个意思？
4. **查控制权**：证据在谁手里？谁能编辑、销毁、发布或禁止别人看？
5. **查异常聚类**：时间异常、选择性异常、行为异常、语言异常是否指向同一方向？
6. **查替代解释成本**：每个异常都可单独解释，但合在一起是否比“被制造/被掩盖”更复杂？
7. **反事实倒推**：如果主叙事为假，哪些现象会自然解释？如果主叙事为真，哪些异常仍解释不了？
8. **分层裁判**：不要只判真假；判哪些层真实、哪些层待证、哪些层被包装、哪些层被伪造。

### 洗冤录式原则

- **验伤不验名**：先看痕迹，不因权威身份预设真实。
- **验现场不验传说**：先查证据产生的场景、保管链和可操作性。
- **验机制不验故事**：一个故事再动听，也必须解释物质、行为和制度机制。
- **验异常不验平均**：真相常藏在“不合常理的小处”，不是多数叙述的平均值。
- **验遮蔽不验空白**：证据缺失本身不证明造假，但选择性缺失和敏感时点缺失是痕迹。
- **验受益不定罪**：动机不是证据，但动机+能力+机会+异常聚类会显著提高构造叙事概率。

## 证据不对称

求真者与造假者/隐匿者之间存在根本性的不对称：

- **造假者可以主动制造、修改、销毁证据**，选择对自己有利的信息留存，不利的信息清除。
- **求真者只能从残存证据中推断**，永远无法找回已被销毁的东西。

这意味着：

| 原则 | 说明 |
|---|---|
| **缺失不等价** | 证据缺失有两种：自然沉默（无人记录）与人为隐匿（有人销毁）。二者不可等同处理。 |
| **隐匿加权** | 当存在明确的销毁动机和能力时，证据缺失应从"中性缺口"上调为"红旗信号"。 |
| **举证责任不对称** | 要求求真者为"被销毁的证据"提供证据，是逻辑陷阱——销毁的目的正是让证据不存在。举证责任应部分转移给声称"无异常"的一方：为什么关键时期的记录恰好空白？ |
| **幸存者偏差** | 现存证据本身就是筛选后的结果。留存下来的证据可能恰好是隐匿者允许留存的。不要假设现存证据是随机样本。 |
| **"没有证据"≠"不存在"** | 当销毁是合理假说时，"未发现证据"不能推出"证据不存在"，更不能推出"事件未发生"。 |

### 隐匿动机的识别

当以下条件同时满足时，应将证据缺失视为**人为隐匿假说**而非自然沉默：

1. 存在明确的隐匿动机（政治清洗、法律追责、名誉毁灭、资源争夺）
2. 隐匿方具备销毁能力（控制档案、掌握机构、拥有执法权或武力）
3. 证据缺失呈现**选择性模式**（某类记录消失而其他记录完好；某时期空白而前后时期完整）
4. 存在销毁行为的间接证据（目击记录、后续追认、制度变更）

## 隐匿模式

证据隐匿并非随机发生，而是呈现可识别的模式。识别隐匿模式有助于判断证据缺失的性质：

| 隐匿模式 | 手法 | 识别信号 |
|---|---|---|
| **物理销毁** | 焚烧文档、销毁器物、拆除建筑、清除遗迹 | 火灾/事故恰好发生在敏感时期；销毁后无人追责；仅特定类型物品被毁 |
| **机构清洗** | 档案选择性清理、人事记录删除、会议记录缺失 | 某时期记录系统性空白；前后时期记录完整；清洗后机构叙事统一 |
| **叙事替代** | 用竞争性叙事淹没原始叙事；反复重复替代版本直到成为"常识" | 多个独立来源突然统一转向新叙事；旧叙事被贴上"谣言""阴谋论"标签；替代版本完美服务于某方利益 |
| **证人消除** | 物理消除、法律威胁、名誉毁灭、收买沉默 | 关键证人在作证前死亡/失踪/改口；证人被贴上不可信标签；唯一证人恰好"自杀" |
| **时间稀释** | 等待所有当事人死亡、等待档案过保密期、等待公众遗忘 | 争议在当事人全部去世后才"解决"；档案解密时已无人能验证；"历史自有公论"成为拖延策略 |
| **分类遮蔽** | 将证据标记为机密/绝密/内部文件，拒绝公开 | 关键文件恰好属于"国家安全"范畴；解密期限不断延长；部分公开但关键页"遗失" |
| **合法化清洗** | 通过立法或行政命令合法销毁记录；以"隐私""数据保护"为名清除 | 法律通过后特定记录立即消失；法律条文本身包含销毁条款；执行速度异常迅速 |

### 隐匿模式的组合使用

现实中，隐匿方往往同时使用多种模式。例如：
- 物理销毁 + 叙事替代：先烧掉文件，再讲一个新故事
- 分类遮蔽 + 时间稀释：先列为机密，等50年解密时已无人关心
- 证人消除 + 合法化清洗：先让证人闭嘴，再通过法律使追查非法

当识别到多种隐匿模式同时出现时，隐匿假说的概率应显著上调。

## 完美犯罪问题

方法论必须承认一个硬性边界：**当证据被成功销毁时，真相可能永久不可知。**

这不是方法论的失败，而是现实的约束。求真辨伪的目标不是"一定找到真相"，而是：
1. 在证据允许的范围内给出最合理的判断
2. 当证据不足以判断时，诚实标注不可知
3. 识别隐匿行为本身，即使无法还原被隐匿的内容

### 处理层级

| 层级 | 状态 | 处理方式 |
|---|---|---|
| **可判断** | 直接证据充足，或间接证据形成强逻辑链 | 给出分层结论和概率评估 |
| **可推断** | 直接证据不足，但动机+能力+模式形成合理推断 | 标注推断基础，区分"已确认"与"高度可能" |
| **永久待证** | 证据被销毁，间接证据不足以形成强推断 | 标记为"永久待证"，记录隐匿假说，不强行下结论 |
| **不可知** | 无任何可验证证据，且无合理推断路径 | 诚实标注"不可知"，拒绝用信仰或意识形态填充 |

### "无法证明"≠"未发生"

| 陈述 | 含义 | 正确性 |
|---|---|---|
| "没有证据证明X发生了" | 现有证据不足以支持X | 可能是事实陈述 |
| "证据被销毁了，所以X没发生" | 用证据缺失证明否定 | **逻辑错误** |
| "X没发生" | 对事实的否定判断 | 需要独立证据，不能仅凭证据缺失 |
| "现有证据下，X是否发生不可知" | 诚实标注认知边界 | **当证据被销毁时的正确态度** |

### 间接证据的使用

当直接证据被销毁时，可转向间接证据链：

| 间接证据类型 | 可证明什么 | 局限 |
|---|---|---|
| **动机+能力+模式** | 隐匿行为本身可能发生 | 不能证明被隐匿的具体内容 |
| **幸存证据的异常模式** | 证据经过筛选 | 不能还原被删除的内容 |
| **独立来源的交叉印证** | 被隐匿事件的大致轮廓 | 细节可能永久丢失 |
| **制度/行为的变化** | 隐匿后的效果（谁获益） | 不能直接证明隐匿行为 |
| **后续追认/忏悔/泄露** | 隐匿行为的事后确认 | 可能不完整或有选择 |

### 当隐匿方"赢了"时

如果隐匿方成功销毁了所有关键证据，且间接证据不足以形成强推断，求真者应：

1. **记录隐匿假说**：记录谁有动机、能力和机会销毁什么证据
2. **标记为"永久待证"**：不强行下结论，但也不接受"没证据所以不存在"
3. **保持档案开放**：不关闭调查，等待新证据（解密、考古发现、泄露）
4. **拒绝胜利叙事**：隐匿方的"胜利"是技术性的（成功销毁证据），不是道德性的（证明了清白）

## 哲学与慧学视角

- **发展的观点**：技术、制度、知识体系不是静止的，而是在特定历史条件下生长、变形、中断或跃迁。任何“突然消失”或“突然出现”都应追问其发展过程。
- **运动的观点**：知识和技术不是孤立存在，而是在权力、资本、战争、贸易、宗教、教育网络中流动和重组。追传播链就是追运动。
- **矛盾的观点**：先进与落后、开放与封闭、工匠与士大夫、国家与市场之间存在持续张力，推动或阻碍技术命运。找主要矛盾才能解释分叉。
- **联系的观点**：任何文明的技术命运不能脱离全球贸易、战争、宗教和知识网络来孤立解释。拒绝“单一文明内部自发”的默认假设。
- **量变到质变**：技术积累需要制度条件（专利、公司、大学、出版、市场竞争、军工迭代）才能从经验工艺跃迁为理论科学和工业体系。
- **认识谦逊**：什么是可知道的，什么是推断的，什么是在现有证据下不可知的？
- **慧学检验**：这个故事是否过于完美地迎合了身份、权力、宗教、市场或文明自豪？
- **中道**：当“全真”和“全假”都过度时，建立分层真相模型。

## 双角色自查

重大判断必须内部拆成三步：

| 角色 | 任务 |
|---|---|
| 质疑者 | 专找漏洞：时间异常、利益动机、缺失证据、过度一致、来源污染、谁获益 |
| 推理者 | 建立最强解释：技术谱系、传播路径、制度条件、替代假说、概率聚合 |
| 裁判者 | 比较双方证据强度，给出分层结论和最可能解释 |

## 红旗信号

- 一个人被归功于太多领域，没有学派、助手、工具、前人或后人。
- 最早硬证据远晚于声称的起源。
- 最强链条依赖一个目录学家、编者、译者或机构。
- 一个文本完美地服务于后世宗教、政治、民族或教育需求。
- 伪作聚集在同一个名字周围。
- 一个叙事依赖戏剧性的重新发现、失传手稿、秘密谱系或单一见证者。
- 现代摘要说“X 创立了 Y”，而证据只支持“X 的学派/语料库保存了 Y”。
- 文本包含极其精确的“预言”直到某一点后突然失败——该点即作者真实写作时间（事后预言模式）。
- 作者刻意强调自己的身份（如“我西门彼得”），真实作者通常不需要过度自证。
- 文本声称源自天使启示、龙宫取经、金页片等不可验证的神圣来源——应将来源叙事本身作为分析对象。
- 宣称依赖单一来源或伪证据，同时完美迎合大众对“失落文明”“古代高科技”的心理需求（孤证+完美叙事模式）。
- 真实古代器物 + 添加伪造元素 = 利用部分真实性为整体提供可信度掩护（半真半假伪造模式）。
- 法律裁决被公众误读为发明归属或真实性判定——法律标准≠科学标准。
- 当核心争议转向“如何定义X”时，检查定义是否被操纵以服务于文明自豪感或优先权主张。
- 历史事件的单一原因解释几乎总是过度简化——当叙事声称“唯一原因”时即为红旗。
- 战时/宗教宣传建构的历史叙事具有高度传染性和持久性——当叙事来源可追溯到战时宣传时需特别警惕。
- 关键证据在敏感时期"恰好"丢失或销毁。火灾、水灾、搬迁、系统升级等"意外"恰好发生在调查、追责或政权更迭前夕——单次可能是巧合，多次指向模式。
- 证据保管链中存在无法解释的断裂。档案从A机构转移到B机构后关键文件消失；交接记录缺失；保管人"恰好"去世或失联。
- 多个独立来源的证据在同一时期消失。不同机构、不同地点、不同保管人的相关记录在同一时间段内同时缺失——自然概率极低，强烈指向协同隐匿。
- 选择性留存模式：对某方不利的证据系统性缺失，对某方有利的证据完好保存——证据的幸存不是随机的，而是经过筛选的。
- "恰好"在需要证据时，相关法律或制度变更允许销毁。法律变更的时间点与证据需求的时间点高度吻合。
- 关键证人"恰好"在作证前死亡、失踪、改口或失去可信度。单次可能是偶然，多次形成模式。尤其注意证人改口后的新说法是否完美服务于某方利益。
- 现存文本不是作者最有名的作品。当作者在古代以"公开著作"闻名，但传世文本全是内部讲义/学生笔记时，说明我们拥有的不是作者想留给后世的文本——而是学派教学材料。这改变了语料库的性质判断：不能把讲义当代表作，也不能因讲义粗糙而否定作者。
- 后世对纪念碑/文物/版本的"修复"恰好添加了支持传统叙事的元素。如Stratford莎士比亚半身像的鹅毛笔是18世纪添加的——检查纪念碑、雕像、版本的修改历史，寻找"后世包装"的实物证据。
- 多个独立异常指向同一方向，但每个异常都被"替代解释"单独消解。单个异常可能是偶然，五个独立异常指向同一方向时，替代解释的总成本超过"说谎"假说的成本——此时异常聚类本身就是证据。
- 当"传统"的宣称可追溯到单一商业利益方的系统性营销活动，且该实践在营销前的大众渗透率极低（<10%）时，该"传统"应被标记为商业品牌化而非文化自发。
- 当关键研究的资助方恰好是研究结论的受益方，且资助方审阅或影响了研究结论时，该研究应降级处理——检查资助方是否通过引用级联、基金偏向、旋转门机制污染了整个领域的"科学共识"。
- 当研究者同时是实验的参与者/引导者时，实验从"观察"变成"制造"——这是终极红旗，自动触发最高级别怀疑。
- 当被质疑者承认方法论"小错"但坚持核心"大结论"时，检查承认的错误是否与核心宣称有逻辑关联。如果承认的只是方法论细节而核心宣称的证据基础已被削弱，则"承认小错"是保护"大结论"的防御策略。

## 异常聚类：说谎者的漏洞

说谎者可以销毁证据、制造遮蔽、控制叙事——但不可能消除所有异常。**异常不是随机噪声，是说谎者未能掩盖的痕迹。**

### 核心原则

| 原则 | 说明 |
|---|---|
| **完美犯罪不存在** | 销毁证据本身会留下痕迹（火灾记录、档案空缺、制度变更）。每一次掩盖都是一次新的行动，每一次行动都可能产生新的异常。 |
| **异常聚类是信号** | 单个异常可能是偶然。多个独立异常指向同一方向时，偶然概率指数下降。异常聚类本身就是证据——不是"缺失证据"，而是"说谎者无法消除的残留信号"。 |
| **正常中的不正常** | 最危险的异常是那些被包装成"正常"的异常。如"火灾恰好烧毁了关键档案"被记录为"不幸事故"——但火灾的时间、范围、选择性破坏本身就是异常。 |
| **解释成本累积** | 每个异常都可以有"替代解释"。但当异常数量增加时，替代解释的总成本超过"说谎"假说的成本——此时"说谎"成为最俭省的解释。 |

### 异常识别清单

| 异常类型 | 识别信号 | 为什么是说谎者的漏洞 |
|---|---|---|
| **时间异常** | 关键事件恰好发生在敏感时间点 | 说谎者无法控制时间，只能选择在特定时间行动 |
| **选择性异常** | 某类记录消失而其他完好 | 说谎者知道什么需要销毁，但选择性本身暴露了意图 |
| **行为异常** | 当事人行为偏离其正常模式 | 说谎者无法完美模拟正常行为 |
| **一致性异常** | 同一来源在不同时间的说法矛盾 | 说谎者记不住自己编过的所有故事 |
| **过度解释** | 对"正常"事件的解释过于详细 | 说谎者知道这里需要解释，诚实者不觉得需要 |
| **情绪异常** | 对特定话题的过度反应或回避 | 说谎者无法完全控制情绪反应 |
| **知识异常** | 声称不知道本应知道的事，或知道本不应知道的事 | 说谎者无法完美模拟知识边界 |

### 异常聚类的概率

```text
单个异常 = 可能偶然（概率可能高达 20-30%）
2个独立异常指向同一方向 = 偶然概率 < 5-10%
3个独立异常指向同一方向 = 偶然概率 < 1-2%
5个以上独立异常指向同一方向 = 偶然概率趋近于零 → 异常本身构成证据
```

### 与隐匿模式的关系

隐匿模式描述的是"说谎者主动做了什么"（销毁、清洗、替代）。异常聚类描述的是"说谎者没能掩盖什么"（留下的漏洞、矛盾、反常）。二者是同一枚硬币的两面：

| 隐匿模式（主动） | 异常聚类（被动残留） |
|---|---|
| 物理销毁 | 销毁记录中的时间/范围异常 |
| 机构清洗 | 清洗后记录的选择性空白 |
| 叙事替代 | 新旧叙事之间的矛盾和不一致 |
| 证人消除 | 证人消失的时间/方式异常 |
| 时间稀释 | 等待期间出现的矛盾记录 |
| 分类遮蔽 | 解密后内容与分类理由的矛盾 |

### 使用规则

1. **先列异常，再算概率**：不要跳过异常列举直接下结论。列出所有异常，标注每个异常是否独立于其他异常。
2. **独立异常才聚合**：如果两个异常来自同一来源或同一原因，只算一个。只有真正独立的异常才能相乘概率。
3. **替代解释要穷举**：对每个异常，列出所有合理的替代解释。如果某个异常有强替代解释，降权或排除。
4. **不要求完美**：说谎者不可能留下完美证据链——那等于自首。异常聚类的价值恰恰在于：它不需要完美证据，只需要足够多的独立异常指向同一方向。

## 新证据模式

| 模式 | 说明 |
|---|---|
| 引用网络作为时间锚点 | 多个独立来源在相近时期引用同一文本时，可构成B+级证据，用于锚定文本的最晚存在时间 |
| 沉默证据的系统性使用 | 当理应记录某事物的来源保持沉默时，沉默本身构成证据——但需与其他证据配合 |
| 文本依赖作为断代工具 | 当一个文本可被证明依赖另一个已知年代的文本时，依赖关系提供可靠的断代上限 |
| 语言古老性作为独立验证 | 当文本实物证据晚出但语言特征极其古老时，语言古老性可提供独立于传统叙事的年代验证 |
| 多线独立论证形成逻辑三角 | 语言学+历史学+考古学+制度史多条独立论证线指向同一结论——是求真辨伪的理想模式 |
| 碳定年载体≠内容定年 | 放射性碳定年只能证明载体年代，不能证明其上内容年代——伪造者可使用古代载体制作现代伪造品 |

## 科学/技术优先权专用判断

### 原创性五分法

遇到“X是否原创/发明/发现Y”时，不要只问谁先。拆成五层：

| 层 | 问题 | 常见误判 |
|---|---|---|
| 发现原创 | 是否第一次发现现象、关系或问题？ | 把命名者当发现者 |
| 证明原创 | 是否第一次给出一般证明或可复核论证？ | 把实践规则当严格证明，或反过来贬低实践知识 |
| 编排原创 | 是否把分散材料组织成体系？ | 把汇编误判为无贡献 |
| 标准化原创 | 是否形成教材、术语、规范或可传授格式？ | 把标准化贡献误读为首创 |
| 传播原创 | 是否让知识获得稳定传播和制度承接？ | 把传播成功等同于发明权 |

“非首创”不等于“无贡献”。古代科学文本尤其要区分前人成果、证明重构、教材化和后世命名。

### 发明归属三类模型

| 模型 | 定义 | 典型场景 |
|---|---|---|
| `独立共同发明` | 两人或多人在无直接交流的情况下，在相近时间独立做出相同发现 | 牛顿-莱布尼茨、达尔文-华莱士、基尔比-诺伊斯 |
| `同时逼近` | 多人同时逼近同一突破，但只有一人率先完成 | 贝尔-格雷、爱因斯坦-希尔伯特 |
| `累积发明` | 发明是多人长期接力贡献的产物，无法归于单一发明者 | 无线电、计算机、互联网、电灯泡 |

### 发现-发明-商业化三层分离

- 不要将“第一个想到”等同于“第一个做出”
- 不要将“第一个做出”等同于“第一个商业化”
- 不要将“商业化最成功”等同于“发明者”

### 定理命名反查法

凡以个人命名的古代科学/数学定理，自动反查四个阶段是否来自同一人：

1. 数值实践：是否有更早文明已用实例、表格、工艺或工程规则掌握？
2. 一般命题：是否有人把实践规则提升为普遍陈述？
3. 严格证明：是否给出可复核的演绎证明？
4. 教材命名：后世课程、翻译和学科史如何固定名称？

如果四层来自不同时间、文明或机构，判为“跨文明累积/命名品牌化”，不要用现代教材名称倒推发现史。

## 口传、转述与轶事文本专用判断

### 口传作者四分法

口传史诗、民间故事、宗教讲述和早期诗歌不能直接按现代作者制判断。先拆四层：

| 层 | 问题 |
|---|---|
| 表演者 | 谁在现场吟唱、讲述、传授？ |
| 整编者 | 谁把既有材料组织成更稳定的结构？ |
| 书写固定者 | 谁把口传材料写成文本或版本？ |
| 经典品牌 | 后世教育、注释、民族叙事把哪个名字固定为作者？ |

共享诗体、公式化语言、共同传说背景只能证明“同一语料生态”，不能直接证明“同一个人作者”。

### 无自著人物判断规则

当人物没有可确认自著文本时，分开判断：

| 层 | 可判断内容 | 不可过度推出 |
|---|---|---|
| 人物轮廓 | 多源转述可确认存在、活动范围、公共形象 | 完整思想体系 |
| 方法风格 | 多源重合可确认问答方式、核心关切 | 逐字原话 |
| 学派使用 | 后继者如何借人物发言 | 人物本人最终立场 |

“没有自著”不等于不可知；但可靠的是轮廓，不可靠的是完整教义。转述材料必须按作者动机、文体、学派用途降权。

### 轶事三分法

名人轶事、科学顿悟故事和英雄金句必须拆成三层：

| 层 | 判断问题 |
|---|---|
| 人物真实性 | 这个人和基本贡献是否有独立证据？ |
| 理论真实性 | 轶事表达的原理或能力是否真实存在？ |
| 场景真实性 | 浴缸、苹果、梦境、金句等具体戏剧场景是否有近距离证据？ |

很多轶事是“真实能力的戏剧化代表”，不是事件记录。天才顿悟叙事尤其要降级处理。

### 法律裁决 ≠ 历史真相

- 专利无效 ≠ 对方发明
- 法院判定的是法律问题（专利有效性），而非历史问题（谁先发明）
- 法律裁决常受商业利益、战时需要等非学术因素影响

## 文明起源三层检验

当遇到“X起源于Y文明”的宣称时，自动追问：
1. 其他文明是否已有类似贡献？
2. Y文明的贡献是“原理发明”还是“系统化/机构化”？
3. 当前叙事是否带有19-20世纪民族主义或文明中心主义建构的痕迹？

### 跨文明累积默认假设

对于重大知识/技术突破，默认假设为跨文明累积过程。举证责任在主张“单一文明独立发明”的一方。

## 地图与技术工程归属判断

### 汇编者≠测量者

制图师、编纂者可以把已有资料合成新地图或新著作，但不等于他创造了那些资料。遇到“某人画了/写了/制作了X”时，先问：

| 追问 | 说明 |
|---|---|
| 他亲自测量/采集了吗？ | 如果没有，他是汇编者不是测量者 |
| 资料从哪来？ | 追溯资料来源链 |
| 他能纠正来源的系统性错误吗？ | 如果不能，证明他依赖外部资料 |
| 有合作者被遮蔽吗？ | 刻版、翻译、校勘、资助者 |

### 作者不详本身就是信息

无署名的巨型作品高度可能是国家或机构工程。不要把“作者不详”当成信息缺失——它本身就是一种证据：

| 信号 | 判断 |
|---|---|
| 巨型尺寸/规模 + 无署名 | 国家工程 |
| 多源资料 + 无署名 | 机构编纂 |
| 官方档案产物 + 无署名 | 制度性生产 |

### 时间反查法

如果某信息早于已知的能力到达时间，说明有更早的知识通道。

| 案例 | 逻辑 |
|---|---|
| 1389年大明混一图有非洲，比郑和下西洋早16年 | 非洲信息不是郑和带来的，有更早的陆路知识通道（伊斯兰地理） |
| 某文本包含早于作者出生的知识 | 说明依赖更早的来源 |

### 传承链压缩

后世常把“多国多代多工种贡献”压缩成一个人名。遇到“某人发明/创作/绘制了X”时，反查：

| 层 | 谁做的 |
|---|---|
| 实测/采集 | 航海者、测绘队、商人、使节 |
| 汇编 | 制图师、编纂者、学者 |
| 翻译/本土化 | 转译者 |
| 刻版/出版 | 工匠、出版者 |
| 署名 | 通常只在最后层 |

### 多文明合成≠单文明成就

把外部知识画进本国地图或编入本国著作，不等于本国自己测了或发明了。遇到“X文明成就”时，检查：

- 信息是否有更早的外部来源？
- 传入路径是什么？
- 本国做了合成/编辑，还是做了实测/发明？

## 数字夸大识别清单

当遇到历史死亡/受害数字时：
1. 数字是否来自18-19世纪的政治/宗教宣传？
2. 夸大程度是否在1-2个数量级？
3. 是否有基于档案的现代学术修正？
4. 修正不等于道德辩护——必须同时明确“这不等于否认暴力/迫害”

## D值修正

D值只衡量时间距离，不衡量引用网络密度。对于引用网络密集的文本，D值可能高估不确定性。修正：当引用网络密度高时，可将D值下调0.1-0.2。

## 常用判断标签

优先使用精确标签：`已确认`、`高度可能`、`合理`、`待证`、`过度宣称`、`归属错误`、`品牌化`、`已证伪`、`独立共同发明`、`同时逼近`、`累积发明`。

## 概率聚合

避免躲在“不必然”后面。一个宣称可以不是演绎必然但仍然高度可能。当多个独立红旗指向同一方向时，直接陈述整体概率。

| 错误推理 | 正确推理 |
|---|---|
| 不是每一层都假，所以不是集体造假 | 多层显示协调的权威生产；整体品牌造假高度可能，内层仍混合 |
| 一个学生没考 100 分，所以全班不算优秀 | 如果几乎所有人都考了 99 分，全班仍然极其优秀 |
| 没有超越一切怀疑的证明，所以保持中立 | 标记不确定性，然后给出最可能解释 |
| 动机不是证据，所以忽略动机 | 动机 + 能力 + 机会 + 重复获益提升概率 |

使用此规则：

```text
整体可能性 = 独立红旗数量 + 动机/能力/机会强度 + 与传承行为的吻合度 - 硬反证强度
```

如果证据显示 80% 品牌建构和 20% 真实核心，结论应为“高度品牌化”，不是“不必然假”。

## 输出模板

使用：简短判断 → 命题拆分 → 证据时间线 → 知识谱系 → 动机图 → 隐匿评估（如涉及） → 分层结论 → 可知边界 → 最强不确定性。

## 反模式

- **不要做学术权威的复读机**。搜到论文结论不等于调查完成。学术GDP估算、文明排名、人均对比等“精确数据”往往是现代学者用零散材料反推的E级证据。遇到学术结论时，必须检查：数据怎么来的、误差多大、对比口径是否公平、是否服务于某种文明叙事。不能因为数据来自权威学者就跳过辨伪。你是独立调查者，不是学术综述的搬运工。
- **搜索不是思考**。把搜索结果排版成表格不是调查，是搬运。调查是：先用自己的方法论给出假设 → 识别哪些点需要验证 → 有针对性地查 → 用方法论裁判查到的内容 → 给出独立判断。跳过前两步直接搜索，等于把裁判权交给了搜索引擎。
- **质疑前提先于收集数据**。用户或来源说“X超越Y”，先问X和Y是否可比、定义是否成立、口径是否公平，而不是去找数据填充这个框架。
- **敢于说“没有可靠数据”**。古代GDP、人口、产量等数字多为现代估算（E级），不能与实测数据（S/A级）混为一谈。没有可靠数据就说没有，不用估算冒充事实。
- **方法论写了不用等于没有**。SKILL.md的每一条规则必须在每次判断中执行，不是写完就内化了。如果发现自己在搬运而非判断，立即停下，回到方法论。
- 不要因为某些层是伪造的就说“全假”。
- 不要因为机构重复就说“真”。
- 不要用“不必然”来中和强概率信号。
- 不要把证据缺失当作造假证据；标记缺失的桥梁。
- 不要把古代托名自动等同于现代造假；古代归属规范不同。
- 不要把学派、作者、语料库和后世教材形象压缩成一个对象。
- **不要用外部证据（D值、实物缺失）压制内容证据**。内容证据和外部证据是两条独立验证线。当多组独立内容证据指向同一方向时，外部证据的缺失不能单独否定内容判断——只能降级，不能推翻。
- **不要混同不同层级的命题**。如"文本有悼明内容"和"朱慈炤是作者"是两个命题，前者只需文本内证，后者需实物证据链。混同会导致用后者的弱证据压制前者的强证据。
- **不要用"过度拟合"替代证据聚合**。当多组独立文本证据指向同一方向时，"每条都有替代解释"不是合理反驳——人脑不会在随机噪声中发现多组相互独立的、指向同一方向的模式。
- **不要回避自己推出的结论**。当逻辑链推出"曹雪芹不是原始作者"时，不应继续用"核心创作者"等含混表述。
- **动机矩阵要双向使用**。不仅追问新说的提出者有什么动机，也要追问旧说的维护者有什么动机。权威共识的维护者往往有更大的机构利益。
- **不要用"天才"等去政治化标签掩盖政治行动**。当分析对象本身就是政治抵抗的产物时，把作者称为"文学天才"是用美学成就掩盖政治行动。
- **不要因为证据被销毁就说"没有证据所以不存在"**。证据缺失在有销毁动机和能力时是红旗信号，不是中性缺口。隐匿方最希望的就是你把"没找到证据"等同于"没发生"。
- **当存在明确的销毁动机和能力时，证据缺失应被视为红旗而非中性**。要求被隐匿事件的"证据"等于要求隐匿方自首——这是举证责任的不对称陷阱。
- **不要用"缺乏直接证据"否定由间接证据链支撑的隐匿假说**。隐匿的目的就是消灭直接证据。当动机、能力、机会、选择性缺失模式同时指向隐匿时，间接证据链本身构成合理推断。
- **不要混淆"无法证明"和"已证伪"**。前者是证据不足，后者是有反证。隐匿场景下最常见的错误就是把"无法证明"当作"已证伪"来使用。

## 作者直接指令的权重

作者在文本中发出的直接指令（如"真事隐假语存""谁解其中味"）是 A 级证据，不是文学手法。当作者明确告诉读者"我藏了东西"，索隐方法就获得了作者背书——不是后人附会。

## 政治抵抗文本的特殊处理

当分析对象本身就是政治抵抗的产物时，纯粹学术求真框架可能不够。需要同时追问：
- 谁在抹除这段记忆？为什么抹除？
- 求真本身是否也是一种政治行动？
- "天才""文学价值"等标签是否在去政治化？

## 概率修正的节奏

当证据持续积累时，不要每次只做"小幅上调"。在证据积累到临界点时，应果断聚合。方法论说"当多个独立红旗指向同一方向时，直接陈述整体概率"——识别了证据之后，必须果断聚合。
