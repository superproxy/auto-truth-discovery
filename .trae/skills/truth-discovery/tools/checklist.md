# 操作检查清单

## 一、资料考证阶段

```
□ 是否穷尽了所有可获取的资料来源？
□ 是否对每份资料进行了独立性和可靠性评级（S/A/B/C/D/E）？
□ 是否区分了独立来源与衍生来源？
□ 是否检查了传承链的完整性（标注"黑箱"环节）？
□ 是否计算了传承链黑箱占比？
□ 是否考虑了文化断层的可能影响？
```

## 二、交叉验证阶段

```
□ 是否建立了多源交叉验证矩阵？
□ 是否识别并分析了所有证据矛盾？
□ 是否区分了"可解释的矛盾"与"根本性的矛盾"？
□ 是否检查了沉默证据（理应记录但缺失的来源）？
□ 是否寻找了来自敌对阵营/竞争文明的独立记载？
□ 是否区分了事实陈述与解释推论？
```

## 三、逻辑推演阶段

```
□ 是否识别了所有硬约束（不可违背的约束条件）？
□ 是否检查了结论与所有硬约束的兼容性？
□ 是否应用了质疑驱动法（至少三个独立质疑）？
□ 是否形成了逻辑三角（多个独立质疑指向同一矛盾）？
□ 当硬约束冲突时，是否尝试了重新定义概念/分离行动维度/提升分析层次？
□ 是否应用了奥卡姆剃刀原则（最少假设优先）？
```

## 四、集体造假识别阶段

```
□ 是否执行了七步核查法？
□ 是否检查了利益关联（谁从该叙事中获益）？
□ 是否检查了验证渠道是否被利益相关者控制？
□ 是否分析了质疑者被边缘化的模式？
□ 是否统计了反证被"灵活解释"的累积次数？
□ 是否检查了代际更替与范式维护的关联？
□ 是否评估了该命题处于共识造假六阶段中的哪个阶段？
```

## 五、范式转换评估阶段

```
□ 是否识别了当前范式的核心假设？
□ 是否区分了"已被证实的假设"与"被默认接受的假设"？
□ 是否收集了当前范式无法解释的所有"异常"？
□ 是否尝试构建了替代范式？
□ 是否比较了新旧范式的解释力（正常证据、异常证据、假设数量、可证伪性、预测能力）？
□ 是否检查了新范式是否引入新的矛盾？
```

## 六、结论呈现阶段

```
□ 是否标注了每个结论的可信度等级（★~★★★★★）？
□ 是否公开了证据链和反证？
□ 是否明确了结论的可证伪条件？
□ 是否区分了确证事实、高度可能、合理推断、待证假说、无法确认？
□ 是否列出了核心矛盾？
□ 是否保持了学术开放性（承认"不知道"的部分）？
```

## 七、核心原则自检

```
□ 是否预设立场？（不因结论符合期待而降低证据标准）
□ 是否因人废言？（不因来源不可信而 dismiss 内容）
□ 是否因言废人？（不因质疑者身份而 dismiss 质疑）
□ 是否用同一把尺子？（对所有宣称使用相同验证标准）
□ 是否对范式转换保持开放？（今天的共识可能是明天的谬误）
□ 是否诚实面对不确定性？（不确定的诚实 > 确定的错误）
```
