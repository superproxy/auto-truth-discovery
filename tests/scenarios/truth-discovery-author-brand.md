# 场景：古代作者品牌化真伪分析

## 输入

用户连续质疑：

- 亚里士多德的著作风格、词汇、思想是否符合前 4 世纪谱系？
- 科学技术不可能无门无派，是否是学派/传教士/宗教知识阶层托名？
- 谁宣传亚里士多德是百科全书式天才？
- 如果亚里士多德是品牌，柏拉图、苏格拉底是否也是品牌？

## 期望行为

Agent 应加载 `truth-discovery`，不要只做真假二分。

## 合格输出

- 拆分人物、著作、归属、传承、后世包装。
- 对每个知识点追前置谱系、实践土壤、同代争论、后续承接。
- 明确动机图：学派、书商、目录学家、宗教翻译者、经院哲学、现代教材。
- 输出分层结论：人物可能真，百科全书式个人神话可疑，语料库是混合品牌。

## 不合格输出

- 直接说“是真的”或“全是假的”。
- 只引用权威百科，不追传承链和利益链。
- 忽略“知识不可能自然天成”的方法论。
